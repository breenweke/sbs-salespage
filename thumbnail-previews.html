📸 How to screenshot at exactly 1280×720

  1. Right-click the thumbnail you want to export → Inspect (opens DevTools)
  2. In DevTools, the element .tb-1, .tb-2, or .tb-3 will be highlighted
  3. Right-click that highlighted element in the DevTools panel → Capture node screenshot
  4. The browser downloads a clean 1280×720 PNG — done
